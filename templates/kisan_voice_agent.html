⚙️ VAPI Voice Workflow Setup Instructions

  1. Go to dashboard.vapi.ai and sign in.
  2. Go to Assistants and click Create Assistant. Name it "Kisan Mitra".
  3. System Prompt Configuration: Paste the exact prompt designed for Kisan Mitra (see documentation). Set it to reply ONLY in short, natural Hindi sentences without bullet points.
  4. Add Keywords: In the Advanced settings, add keywords like: disease, pest, fertilizer, irrigation, sowing to help the LLM route topics quickly.
  5. Select Voice: Go to the Voice tab, choose a Hindi-capable voice like "ElevenLabs - Rachel (or equivalent Hindi male voice)".
  6. Copy your Public API Key (from Settings) and Assistant ID, and paste them in the JavaScript section of this file (bottom of the code).
